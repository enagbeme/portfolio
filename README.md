# Enoch's Personal Portfolio

This repository contains Enoch's personal portfolio website, showcasing his skills, education, work experience, and projects. The website is built using Next.js with TypeScript, providing a modern and performant user experience.

## Technologies Used:
- **Next.js:** A React framework for building fast, scalable, and SEO-friendly web applications.
- **TypeScript:** A superset of JavaScript that adds static typing to the language, enhancing code quality and maintainability.
- **React:** A JavaScript library for building user interfaces.
- **CSS:** For styling and responsive design.

## Features:
- **Responsive Design:** Adapts seamlessly to various screen sizes and devices.
- **Dynamic Content:** Displays personal information, education, skills, work experience, and projects.
- **Client-Side Interactivity:** Includes scroll effects for the header and a scroll-to-top button for enhanced user experience.
- **Static Site Generation (SSG):** Leverages Next.js for optimized performance and fast loading times.
